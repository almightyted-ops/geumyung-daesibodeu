# 📊 내 금융 대시보드

세계 증시 · 주요 뉴스 · 미국/한국 지수 · 섹터 전망 · CNN 공포탐욕지수 · 내 관심종목 차트를
**한 화면에서** 보는 개인용 대시보드입니다. 설치 없이 브라우저로 바로 열립니다.

## ▶ 여는 법
1. `index.html` 파일을 **더블클릭** 합니다. (Chrome / Edge 권장)
2. 인터넷이 연결되어 있어야 차트·뉴스가 표시됩니다.
3. 자주 본다면 브라우저에서 `Ctrl + D` 로 즐겨찾기 하거나,
   `index.html`을 우클릭 → **보내기 → 바탕화면(바로가기 만들기)** 하면 편합니다.

> 위치: `바탕화면 \ 금융대시보드 \ index.html`

## ⭐ 내 관심 종목 바꾸기 (코드 몰라도 됩니다)
`index.html`을 메모장으로 열어, 위쪽 **`const CONFIG = {`** 블록에서 종목만 바꾸세요.

| 구분 | 표기 방법 | 예시 |
|------|-----------|------|
| 미국 종목 | `거래소:티커` | `NASDAQ:AAPL`, `NASDAQ:NVDA`, `NYSE:JPM` |
| 한국 종목 | `KRX:종목코드` | `KRX:005930`(삼성전자), `KRX:000660`(SK하이닉스) |

**자주 쓰는 한국 종목코드**
| 종목 | 코드 | 종목 | 코드 |
|------|------|------|------|
| 삼성전자 | 005930 | NAVER | 035420 |
| SK하이닉스 | 000660 | 카카오 | 035720 |
| LG에너지솔루션 | 373220 | 현대차 | 005380 |
| 삼성바이오로직스 | 207940 | 기아 | 000270 |
| 셀트리온 | 068270 | POSCO홀딩스 | 005490 |

바꿀 곳 예시:
```js
usTickers: ["NASDAQ:AAPL", "NASDAQ:NVDA", "NASDAQ:TSLA"],   // ← 미국
krTickers: ["KRX:005930", "KRX:000660", "KRX:035420"],     // ← 한국
mainChart: "NASDAQ:NVDA",                                   // ← 큰 차트 기본 종목
```
저장 후 브라우저에서 **새로고침(F5)** 하면 반영됩니다.
> 티커를 잘 모를 땐, 대시보드 가운데 **상세 차트** 안에서 직접 종목을 검색해 바꿔도 됩니다.

## 🧭 화면 구성
- **상단 티커 테이프** — 주요 지수·환율·원자재·코인 실시간 흐름
- **시장 심리** — CNN 공포·탐욕 지수 + S&P500 / 코스피 기술적 신호
- **세계 증시** — 시장 개요 + 아메리카·아시아·유럽 주요 지수
- **내 관심 종목** — 상세 차트 + 미국/한국 종목 시세
- **섹터** — S&P500 섹터 히트맵 + 섹터 향후 전망 요약
- **뉴스** — 글로벌 시장 뉴스(TradingView) + **한국 경제 뉴스(연합뉴스)** + 경제 캘린더

## 🚀 바로가기 · 자동 실행
- **바탕화면 바로가기**: `바탕화면 \ 금융 대시보드` 아이콘을 더블클릭하면 바로 열립니다.
- **자동 실행(로그인 시 자동으로 열림)**: 시작프로그램에 등록되어, 컴퓨터 켤 때마다 대시보드가 자동으로 뜹니다.

**자동 실행을 끄고 싶다면** (아래 중 하나):
1. `Win + R` → `shell:startup` 입력 → 열린 폴더에서 **`금융 대시보드.lnk` 삭제**, 또는
2. `Ctrl + Shift + Esc`(작업 관리자) → **시작프로그램** 탭 → "금융 대시보드" → **사용 안 함**

## 🔄 "섹터 향후 전망" 최신화
섹터 전망 텍스트는 자동 갱신이 아닙니다(무료·키 없는 구조라서 그렇습니다).
최신 내용으로 바꾸고 싶으면 Claude Code에게 이렇게 말하세요:

> **"섹터 전망 업데이트해줘"**

그러면 최신 뉴스를 검색해 해당 부분만 다시 작성해 드립니다.

## ℹ️ 참고 / 한계
- **CNN 공포·탐욕 지수**와 **한국 경제 뉴스**는 무료 공개 프록시를 통해 가져옵니다. 프록시가 일시적으로 막히면
  "불러오지 못했습니다" 문구와 직접 보기 링크가 표시됩니다(다시 새로고침하면 대개 복구).
- 차트/데이터 제공: **TradingView** 무료 임베드 위젯. 일부 지수는 15~20분 지연될 수 있습니다.
- 완전 오프라인에서는 위젯이 표시되지 않습니다.
- 본 대시보드는 **정보 제공용**이며 투자 권유가 아닙니다. 투자 판단과 책임은 본인에게 있습니다.
